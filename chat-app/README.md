## Data structure

The following lists the structure of the data:

### `profile`

```js
{
  "name":      <string>,
  "avatarURL": <string>
]
```

### `chat`

```js
{
  "id":       <number>,
  "contact":  <contact>,
  "messages": [<message>, <message>, ...]
}
```

### `contact`

```js
{
  "id":        <number>,
  "name":      <string>,
  "avatarURL": <string>
}
```

### `message`

```js
{
  "id":     <number>,
  "fromMe": <boolean>,
  "date":   <string / date>,
  "text":   <contact>,
  "read":   <boolean>
}
```
