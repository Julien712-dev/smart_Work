## Data structure

The following lists the structure of the data:

### `chats`

```js
[
  <chat>,
  <chat>,
  ...
]
```

### `profile`

```js
{
  "name": <string>,
  "avatarURL": <string>
]
```

### `chat`

```js
{
  "id":       <number>,
  "contact":  <contact>,
  "messages": [<message>, <message>, ...]
}
```

### `message`

```js
{
  "id":     <number>,
  "fromMe": <boolean>,
  "date":   <string / date>,
  "text":   <contact>,
  "read":   <boolean>
}
```
